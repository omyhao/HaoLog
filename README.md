# HaoLog
基于C++17的高性能线程安全日志库
# 特性
- 高效的日期格式化
- 支持固定文件大小滚动
- 日志行使用少内容在栈、多内容在堆的形式
- 支持流式日志格式
- 使用pre-allocated memory来减少内存分配
- 日志行使用move进行消息存取

# 性能
1,000,000条日志，耗时403ms


# 可改进点
